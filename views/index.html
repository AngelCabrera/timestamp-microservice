TimeStamp Microservice

How to fecth:

  1. Make an request to: https://time-microservice-api.glitch.me/api/timestamp/:date_string. Replace ':date_string' with your desired string.
  2. If you just want to get the current time, you can make your request to: https://time-microservice-api.glitch.me/api/timestamp.
  3. Remember to use a valid format string, for example: 2018-01-01, 1514764800

Example Usage:

https://time-microservice-api.glitch.me/api/timestamp/2018-01-01

https://time-microservice-api.glitch.me/api/timestamp/1514764800
